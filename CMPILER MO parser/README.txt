Instructions:

1. compile the C.g4 file, using "java -jar ../lib/antlr-4.9-complete.jar C.g4" command in the terminals IDE.
(The directory should be in the /src.)

2. Run App.java

3. In the upper left side of the UI, there is a menu bar with 'Load' tab. Click the 'Load file' item
to load the preferred input txt file.

3.1. In case when the menu bar is not visible, click the maximize button.

4. Once you open the file, you will be able to see the errors generated by the parser.
